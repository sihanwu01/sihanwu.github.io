ScienceQuiz

Project Description

ScienceQuiz is a non-profit organization that aims to promote science by encouraging students to participate in a series of competitive quiz events. The organization wants to have an online application that tracks and calculates results from the competitions they host. This is a semester long project for the course 67-272: Application Design and Development.

Key Skills

Database Design and Development, MVC, Unit Testing, Website Design, Wireframing, Ruby on Rails
