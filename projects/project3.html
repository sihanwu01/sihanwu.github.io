Bun Bun Bake Shop

This project is built over the course of several assignments from 05-430 Programmable User Interfaces.

The objective of this project is to design a website for a local Pittsburgh bake shop that delivers to nearby customers. The project involved developing low-fi, hi-fi prototypes, as well as creating a working web interface.
